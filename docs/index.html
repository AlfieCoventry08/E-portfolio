Featured Projects

Here is a selection of my recent work showcasing my skills in web development and design.

Low Polly House Model

Here I have modelled a house by using primitive meshes like cubes, cylinders and planes. For this model I started out by making the house and making sure that it was shaped correctly and looked how it was meant to. Then I moved on to adding the details to the house like the door, door handle and chimney. When it came to doing this using Blender's inset tool was very important in the creation, as it allowed me to make some of the shapes look how they did. Similarly the other tools like the extruding tool were also very important in shaping the model and making it a good size.

Health Stim Asset

Initially when I first started to create this asset I started off by looking for inspiration. The inspiration for the asset I made was Call of Duty and the health stim that was from the game. After using the inspiration I started off by using Blender's tools to make the overall shape and the details, like where the needle would come out. After I had completed this I started making the coil as a seperate mesh and then added it to the main asset to complete it.

Cannon Tower

Here I have made a cannon tower that can be seen in many games but the game that I particularly found inspirational was Clash Royale. In the process of creating the 3D assets, I not only used ideas that were taken from games, I also made sure to use my own imagination in the creation process. Especially when it came to the foundation and shape of the actual castle. I ended the asset off with making the cannon and colouring it to make sure it looked like an actual cannon.
